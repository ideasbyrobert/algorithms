Remove Duplicates from Sorted Array · Optimal Solution

One write pointer. Unique values compact forward.

Given a sorted array, remove duplicates in-place so each element appears only once. Return the number of unique elements. Since the array is sorted, duplicates are adjacent — a single forward pass suffices.

Concept

Compare Against Last Written

The write pointer starts at index 1 (the first element is always kept). For each read position, compare against the value at write − 1 — the last confirmed unique. If different, write it and advance; if the same, skip.

Algorithm

Algorithm

write ← 1
        for read in 1..n-1:
        if nums[read] ≠ nums[write - 1]:
        nums[write] = nums[read]
        write += 1
        return write
Edge Cases

Edge Patterns

All identical — write never advances past 1; result length is 1.

Already unique — write tracks read; every element copies to itself.

Runs of duplicates — the write pointer skips entire runs, keeping only the first occurrence.

Complexity

Complexity

Time: O(n) — single pass.
Space: O(1) — two pointers.
