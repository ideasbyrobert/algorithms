Algorithms

Interactive algorithm visualizations, formal proofs, and curriculum maps for LeetCode and interview-style problems. The project is organized around forces and mechanics, so each page teaches not just a solution, but the deeper shape of the problem it belongs to.

Browse arrays, strings, linked lists, matrices, binary search, and more through step-by-step lessons designed to be readable, visual, and mechanically precise.
