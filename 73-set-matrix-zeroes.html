Set Matrix Zeroes · Optimal Solution

Use the matrix itself as a notebook.

A spreadsheet auto-clears entire rows and columns that contain a flagged zero cell. Doing this with a separate copy is trivial but wastes space. The trick: commandeer the first row and first column as marker storage — they record which rows and columns must be zeroed. A second pass applies the markers, and a final pass handles the first row and column themselves. O(m × n) time. O(1) extra space.

Concept

First Row + Column as Markers

Zeroing cells immediately would create cascading false markers. Instead, for each interior zero at [r][c], we defer the work by writing 0 into the first-column cell [r][0] and the first-row cell [0][c]. These act as flags: "row r needs zeroing" and "column c needs zeroing."

Because the markers overwrite the original first row and column, we must save whether those borders originally contained any zero before we start marking.

Algorithm

Four Phases

The matrix itself serves as the marker array. The first row and first column store which rows and columns must be zeroed — no extra space required.

  1. 1

    Flag the edges

    Scan the first row and first column separately. Record whether either originally contains a zero.

  2. 2

    Mark using first row/column

    For each interior cell that is zero, set matrix[row][0] = 0 and matrix[0][col] = 0 as markers.

  3. 3

    Zero the interior

    For each interior cell, if its row marker or column marker is zero, set the cell to zero.

  4. 4

    Zero the edges

    Using the flags from step 1, zero the first row and/or first column if they originally contained a zero.

Edge Cases

Why Save the Flags First?

Understanding boundary behavior is as important as the main algorithm path.

Marker overwrite hazard

The first row and first column are used as markers, so we must flag their original state before writing markers. Processing interior cells first preserves correct flags.

Interior before edges

Zeroing edges last prevents premature erasure of the markers that interior cells still need to read.

Complexity

Complexity

Time: O(m × n) — three passes over the matrix.
Space: O(1) — markers stored in the matrix itself.
