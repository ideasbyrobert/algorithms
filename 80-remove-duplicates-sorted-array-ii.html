Remove Duplicates from Sorted Array II

LeetCode 80 · Lookback-2 Compaction

Given a sorted array, remove duplicates in-place so each element appears at most twice. The key twist from #26: compare against write − 2 instead of write − 1.

Lookback Two Positions

The write pointer starts at 2 (first two elements always kept). For each read position, compare against the value at write − 2. If different, the value can't be a third copy — write it. If equal, it would be a third occurrence — skip it.

Algorithm

write ← 2
for read in 2..n-1:
    if nums[read] ≠ nums[write - 2]:
        nums[write] = nums[read]
        write += 1
return write

Edge Patterns

Three or more copies — the lookback catches the third and beyond; exactly two survive.

All unique — write tracks read; array unchanged.

Array ≤ 2 — always valid; return as-is.

Complexity

Time: O(n) — single pass from index 2.
Space: O(1) — two pointers.
