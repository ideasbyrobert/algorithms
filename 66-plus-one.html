Plus One

LeetCode 66 · Right-to-Left Carry

A non-negative integer is stored as an array of its decimal digits. Add one to the number and return the updated digit array. The key insight: carry propagation only matters when a digit is 9.

Carry Propagation

Start at the rightmost digit and try to add one. If the digit is less than 9, increment it and stop — no carry. If the digit is 9, it becomes 0 and the carry moves left. This chain continues until a digit absorbs the carry or we run out of digits.

The Algorithm

for i from last index down to 0:
    if digits[i] < 9:
        digits[i] += 1
        return digits
    digits[i] = 0

// all digits were 9
return [1, 0, 0, ..., 0]

The All-Nines Case

When every digit is 9, every position carries. The loop ends without returning, meaning the original number was 10k − 1. The result is 10k: a new array one element longer, starting with 1 and filled with zeros.

Complexity

Time: O(n) worst case — when all digits are 9.
Space: O(n) worst case — if the array must expand. Otherwise O(1) extra.
